# Extreme Distance Communications

The communications system shall support **data return from interstellar distances** exceeding 100 AU.

## Adaptations

- Reduced data rate: 160 bps minimum
- Extended DSN tracking passes
- Data compression where applicable
- Prioritized science data per [[SCI-04]]

Signal travel time exceeds **20 hours** each way at current distances. Supports [[MISS-02]].
