# Ring System Imaging

The imaging system shall characterize the **ring systems** of Jupiter and Saturn.

## Objectives

- Ring particle size distribution
- Ring structure and gaps
- Shepherd moon identification
- Spoke features (Saturn)

Special imaging sequences:
1. Back-lit geometry for fine structure
2. High phase angle for dust detection
3. Time-lapse for dynamics

Coordinated with [[IMG-01]] pointing sequences.
