# Science Instrument Suite

The spacecraft shall carry a comprehensive **instrument payload** for planetary and space science.

## Instruments

| Instrument | Purpose | Requirement |
|------------|---------|-------------|
| ISS | Imaging | [[IMG-01]] |
| IRIS | Infrared | [[INST-02]] |
| PPS | Photopolarimeter | UV/visible |
| MAG | Magnetometer | [[FLD-01]] |
| PLS | Plasma | [[FLD-02]] |
| LECP | Charged particles | Radiation |
| CRS | Cosmic rays | [[SCI-04]] |
| PWS | Plasma waves | Radio/plasma |
| RSS | Radio science | Occultation |
| UVS | Ultraviolet | Airglow |

All instruments radiation hardened per [[SC-02]].
